# 커스텀 스킬 (/dot-claude/skills)

| 스킬 | 설명 | 자동 트리거 |
|------|------|-------------|
| `/cleanup` | 문서 최신화, 규칙 위반 코드 정리, 미사용 코드 정리 | 프로젝트 정리/점검 요청 시 |
| `/cleanup 파일명` | 해당 파일만 점검 | 〃 |
| `/refactor 파일명` | 해당 파일 리팩토링 | 리팩토링 요청 시 |
| `/review` | 변경된 파일 코드 리뷰 | 코드 리뷰 요청 시 |
| `/test-gen 파일명` | 해당 파일의 테스트 생성 및 실행 | 테스트 작성 요청 시 |

> 기존 `commands/`에서 `skills/`로 마이그레이션됨. skills는 frontmatter의 `trigger` 조건에 따라 자동 실행될 수 있음.

---

# Shell 스크립트

| 스크립트 | 설명 |
|----------|------|
| `install.sh` | 신규 환경 세팅용. 저장소 파일을 Claude Code 경로에 복사. 기존 파일이 있으면 건너뜀 |
| `update.sh` | 저장소 파일을 Claude Code 경로에 덮어쓰기. 지침 수정 후 반영할 때 사용 |
| `check.sh` | install/update 후 파일이 정상 배치되었는지 확인 |
